This plugin changes the sorting of a catalog (or product category) so that out-of-stock items are always at the end of the list

== Description ==

This plugin has no settings.
Just activate the plugin and check if sold out items have moved to the bottom of the list.


== Installation	 ==

Just activate the plugin and check if sold out items have moved to the bottom of the list.
This plugin has no settings.
